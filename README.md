# Oil Well Region Selection

## Project goal
Build a model to predict oil reserves in three regions and identify the most profitable region for drilling new wells under risk constraints.

## Data
The dataset contains geological features and target product volume for wells in three regions.

## Tech stack
- Python
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Project steps
1. Data loading and validation
2. Exploratory data analysis
3. Linear regression training for each region
4. Break-even volume calculation
5. Profit estimation
6. Bootstrap risk analysis
## Result
Region 1 was selected as the most attractive region because it showed the highest expected profit with an acceptable risk of losses below 2.5%.
